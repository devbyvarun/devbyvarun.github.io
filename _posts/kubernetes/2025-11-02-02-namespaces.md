### **List all namespaces**

```bash
kubectl get namespaces
# or shorthand
kubectl get ns
```

### **View the current namespace for the active context**

```bash
kubectl config view --minify --output 'jsonpath={..namespace}'
```

> If empty, it means the default namespace is being used.

### **Switch (set) default namespace for the current context**

```bash
kubectl config set-context --current --namespace=<namespace-name>
```

### **Run a single command in a specific namespace (without switching)**

```bash
kubectl get pods -n <namespace-name>
```

### 5. **Create a new namespace**

```bash
kubectl create namespace <namespace-name>
```

### 6. **Delete a namespace**

```bash
kubectl delete namespace <namespace-name>

Deleting a namespace deletes all the resources in that namespace
```
